# brain dump

Brain dump is a to-do list sort of tab. It will have a monthly (recurring) to-do list section, and an ad-hoc to-do / reminders / stuff to look into section.

## monthly section
- Contains a list of monthly to-dos, that need to be checked off once a month, and resets every month automatically.
    - hence the list of monthly to-dos need to be stored appropriately
- the check-off should be a tick and grey-out of the item (shifted to the bottom of the list), but it should still be visible in the list

## ad-hoc section
- this will contain ad-hoc notes / to-dos that will be deleted once done 

## UI
- come up with a single word name for this section to add the tab alongside the message, doodle, nugget, mood
- the UI for both sections should be similar to the 2 sections in nuggets
- the same minimal and aesthetic vibe should be maintained in this UI
